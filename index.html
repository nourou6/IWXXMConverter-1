IWXXMConverter

This Tool can help you to convert IWXXM weather file format to TAC (METAR, TAF, SIGMET, SPECI) ones. for more information visit these sites:

  • IWXXM File Format
  • METAR
  • Metart And TAF decode

Setup Project:

1) First run these to batch scripts for creation of realted maven artifacts.
    creare-avxml-maven-artifact.bat
    creare-jweather-maven-artifact.bat
2) Run WXXMConversionApplication main method.
3) Enjoy Project!
